            How to install
            
    
              
  • Gmail (web): Settings → See all settings → General → Signature. Paste with Copy rich.
    • 
              
  • Apple Mail: Mail → Settings → Signatures. Paste, then uncheck “Always match my default message font.”
    • 
              
  • Outlook (web): Settings → Mail → Compose and reply → Email signature. Paste rich.
    • 
              
  • Mobile clients: use the plain-text fallback.
